## StringTools

the tool for quick convert string format,we can Expand with csscript

用于快速转换字符串格式的小工具
可以通过C#脚本进行扩展

Eg: 
 I want to change **a,b,c,d**  to  **"a"-"b"-"c"-"d"** just type:
```
 -->spe , " " -
```

 special characters tips 
  \s =" "
  \t =tab
  \d =""
  \n = \n (next line)
  \r = \r 

 this definition all in script , you can change by yourself
